# Project Setup

## Backend Setup

1. Navigate to the backend folder:
    ```sh
    cd backend
    ```

2. Install dependencies:
    ```sh
    npm install
    ```

3. Start the development server:
    ```sh
    npm run dev
    ```

## Frontend Setup

1. Navigate to the frontend folder:
    ```sh
    cd frontend
    ```

2. Install dependencies:
    ```sh
    npm install
    ```

3. Start the Angular development server:
    ```sh
    ng serve
    ```
